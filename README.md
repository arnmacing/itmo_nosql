# ЛР1: Healthcheck

Простой HTTP-сервис с одним endpoint’ом для проверки доступности сервиса.

## Требования

- Docker + Docker Compose
- GNU Make

> Все настройки проекта берутся **только** из `.env.local`.

```env
# Порт HTTP-сервера, на котором доступен сервис для запросов
APP_PORT=8080
```

Переменная `APP_PORT` используется:

* Docker Compose для подстановки порта в `docker-compose.yml` (через `${APP_PORT}`);
* Приложением, чтобы слушать указанный порт внутри контейнера.

## Запуск

Запуск одной командой:

```bash
make run
```

Проверка, что сервис поднялся:

```bash
curl http://localhost:8080/health
# ожидается: {"status":"ok"}
```
